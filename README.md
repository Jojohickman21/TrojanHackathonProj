# Hackathon Project for the Conservation and Sustaining of the Worl'ds Oceans
We were tasked with meeting one of 16 UN goals for the world and we chose the goal to conserve and sustainably use the oceans, seas and marine resources for sustainable development.

## Project Parts

Our Project is composed of two parts, the Ocean Spill/Trash Classifier and the Ocean Pollution Awareness Minecraft Mod
